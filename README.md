# 附件打包导出

飞书多维表格侧边栏插件，用于将当前视图导出为 ZIP 包。

ZIP 包内包含：

- `导出数据.xlsx`：当前视图中的可见字段和可见记录
- `attachments/`：附件字段中的文件，按唯一文件名平铺保存

## 功能范围

- 只导出当前视图，不导出其它视图
- 只导出当前视图的可见字段，不导出隐藏字段
- 附件统一作为文件打包，不嵌入 Excel
- Excel 中的附件单元格写入相对链接，解压 ZIP 后可点击打开对应附件
- 文件名冲突时自动追加 `-1`、`-2` 等后缀

## 开发

```bash
npm install
npm run dev
```

将本地 Vite 地址添加为多维表格自定义插件即可调试。

## 构建

```bash
npm run build
```

构建产物位于 `dist/`。`package.json` 中已配置：

```json
{
  "output": "dist"
}
```

`vite.config.js` 已配置 `base: './'`，适合静态部署。

## 发布准备

发布到多维表格插件中心时，请提交项目代码和 `dist/` 构建产物。纯前端插件不需要自备长期运行的服务器，但审核和部署方需要能够访问代码和构建产物。

建议插件信息：

- 插件名：附件打包导出
- 类别：导入导出、附件处理、批量处理
- 简短描述：将当前多维表格视图导出为 Excel，并将附件统一打包下载。
